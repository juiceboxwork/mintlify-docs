## Introduction

In Juicebox, you can send emails via a connected email address, or create tasks to integrate a multi-channel outreach strategy with steps like LinkedIn Connection Request, LinkedIn InMail, or a phone call.

If you'd like to utilize sequences across Juicebox, please ensure an email is integrated via the **Integrations** tab on the left-hand menu. For additional information on how to connect an email, plus our email configuration settings, please refer to our [Email Integrations](/email-integrations) article.

## Sequences overview

To navigate to sequences, select the **Sequences** tab on the left-hand menu.

<Frame>
  ![Sequences tab highlighted in the left-hand menu of Juicebox](/images/sequences/sequences-page.png)
</Frame>

From the Sequences page, you can create a new sequence, track overall or individual sequence performance, and view the total number of scheduled steps queued for a certain day.

## Creating a sequence

To create a sequence, select the **New sequence** button in the top-right corner of the Sequences page.

<Frame>
  ![New sequence button in the top-right corner of the Sequences page](/images/sequences/new-sequence-button.png)
</Frame>

When creating a sequence, you can choose to generate one with AI, start from scratch, clone an existing sequence, or use one of the Juicebox provided templates below.

<Frame>
  ![Sequence creation options including AI generation, starting from scratch, cloning, and templates](/images/sequences/sequence-templates.png)
</Frame>

No matter which method you choose, all emails are fully customizable, and within the sequence editor, you'll be able to:

1. Title the sequence
2. Edit the number of steps and each step's schedule cadence
3. Select the step type
4. Choose the sequence sender
5. Customize the subject line
6. Customize the email text
7. Include a signature
8. Include a candidate unsubscribe link
9. Preview and test your sequence

<Frame>
  ![Sequence editor with numbered callouts for each customization option](/images/sequences/sequence-editor.png)
</Frame>

More information on each one of these features is outlined below.

### Title the sequence

After selecting the template you'd like to use, you can click the pencil icon in the top-right corner to give your sequence a name. For example, if you are using the sequence for a specific project, you can title the sequence after the project for ease of access!

<Frame>
  ![Pencil icon for renaming a sequence](/images/sequences/title-sequence.png)
</Frame>

### Edit the number of steps and each step's schedule cadence

From the sequence overview on the left-hand side, you can add or remove a step. In the editor, you can also set each step's schedule cadence to determine when that step should be sent out.

**Add or remove a step**

Click **Add step** to add another step, or hover over a step to reveal the trash can icon and remove.

<Frame>
  ![Add step button and trash can icon for removing a step](/images/sequences/add-remove-step.png)
</Frame>

**Schedule a step**

Above each step, you will be able to schedule the date and time at which that step will be sent out. Step 1 can be sent immediately or scheduled for another date and time. Steps 2\+ can be scheduled for X amount of days, business days, weeks, months after.

<Frame caption="Step 1">
  ![Scheduling options for step 1: send immediately or at a specific date and time](/images/sequences/schedule-step-1.png)
</Frame>

<Frame caption="Steps 2+">
  ![Scheduling options for steps 2 and beyond: days, business days, weeks, or months after the previous step](/images/sequences/schedule-steps-2-plus.png)
</Frame>

### Select the step type

Next to the step number, you can select the type of step you would like to use. Each step can be set up as an email, LinkedIn Connection Request, LinkedIn Message, or Call.

<Frame>
  ![Step type dropdown showing email, LinkedIn Connection Request, LinkedIn Message, and Call options](/images/sequences/step-type.png)
</Frame>

<Warning>
  **Please note:** Only emails will be automated by Juicebox. Any other step will create a task in Juicebox, and when it's time, the sequence will pause until it is marked as completed.
</Warning>

### Choose the sequence sender

Any mailbox integrated across your team's account can be selected from the **From** dropdown in order to identify which email the sequence will come from. For more information on mailboxes, and mailbox access levels, please refer to our [Email Integrations](/email-integrations) article.

<Frame>
  ![From dropdown for selecting the sequence sender mailbox](/images/sequences/sequence-sender.png)
</Frame>

To configure multi-sender for a sequence, you can select different mailboxes at the individual step level. For example, if I want step 1 to come from my mailbox and step 2 to come from my Hiring Manager's email, I would select my mailbox for step 1, and my Hiring Manager's mailbox for step 2.

<Frame>
  ![Selecting different sender mailboxes for individual sequence steps](/images/sequences/multi-sender.png)
</Frame>

Once the sender's mailbox has been switched, you can choose whether emails sent from each individual mailbox should continue within the same thread or start a new one.

If you choose the same thread, all senders will participate in the same email thread and each additional sender in the sequence will be CC'd for awareness. If you choose a new thread, each unique sender will start their own separate email thread with a unique subject line that you can customize.

### Customize the subject line

Change the subject line, and/or add in a CC / BCC email address next to **Subject**.

<Frame>
  ![Subject line field with CC and BCC options](/images/sequences/subject-line.png)
</Frame>

<Tip>
  Personalized subject lines tend to receive higher open rates, so be sure to utilize a variable, like `{{First Name}}`, to grab the candidate's attention!
</Tip>

### Customize the email text

Craft your outreach message by using AI commands, variables, and project details. Templates can also be edited to fit your own tone of voice or formatted to include any bold text, hyperlinks (e.g., scheduling links) and images.

<Frame>
  ![Email editor with AI commands, variables, and formatting options](/images/sequences/email-text.png)
</Frame>

**AI Commands**

AI Commands are custom personalizations for each profile, that will update to include a one or two sentence overview on the candidate's work experience, skills, etc when sent. In the image above, `{Brief mention of the candidate's relevant skills.}` is the AI Command used to help personalize all outreach efforts in bulk.

<Frame>
  ![AI Command options with pre-set examples for work experience and skills](/images/sequences/ai-commands.png)
</Frame>

With AI Commands, you can also choose between pre-set examples, focused on Work Experience or Skills, or write your own. When writing your own AI Command, consider the following: topic, sentence length, and applicability across your candidate pool.

**Snippets**

Snippets are pre-saved templates created by yourself or your team, so you can pull in reusable content across sequence steps.

<Frame>
  ![Snippets menu for inserting pre-saved templates](/images/sequences/snippets.png)
</Frame>

### Include a signature

At the bottom of your step, select a saved signature to further personalize your message. To learn how to create a signature in Juicebox, refer to our [Manage Account](/manage-account) article.

<Frame>
  ![Signature dropdown at the bottom of a sequence step](/images/sequences/signature.png)
</Frame>

### Include a candidate unsubscribe link

Across from Signature, you can select whether or not you want to include an unsubscribe link for the candidate to use when they receive an email and no longer want to be contacted.

<Frame>
  ![Toggle for including a candidate unsubscribe link](/images/sequences/unsubscribe-link.png)
</Frame>

### Preview and test your sequence

Across from the step number, you can preview what the sequence will look like. From here, you can also input your own email, or a teammate's, to officially test the sequence before sending.

<Frame>
  ![Preview button across from the step number](/images/sequences/preview-sequence.png)
</Frame>

<Frame>
  ![Test email input for sending a test sequence to yourself or a teammate](/images/sequences/test-sequence.png)
</Frame>

When you are happy with your sequence, select **Save** at the top of the sequence to exit the screen, and begin your outreach!

<Frame>
  ![AI sequence generation form with fields for job description, company information, and scheduling link](/images/sequences/generate-with-ai.png)

## Editing an existing sequence

Once a sequence is created, it can be starred or pinned for easy access, edited, cloned, or archived at any time. To edit an existing sequence, select "…" to the right of the sequence \> Edit.

<Frame>
  ![Sequence options menu with edit, clone, and archive actions](/images/sequences/edit-sequence.png)
</Frame>

After your sequence has been edited, you can then choose to apply those changes to the candidates already in the sequence or delay those edits for net-new candidates enrolled only.

<Frame>
  ![Dialog for applying sequence edits to existing candidates or only new candidates](/images/sequences/apply-edits.png)
</Frame>

Please note, only the edits made on a candidate's outstanding steps will update. If you made edits to step 1, but step 1 has already been sent out to 5 candidates, those 5 candidates will only receive the edits made to steps 2\+.

## Non-email steps

If you use the Connection Request, LinkedIn InMail, or Call step type, you can only choose the task owner and pre-write the message you want to send to the candidate within the step.

<Frame>
  ![Non-email step configuration with task owner selection and message field](/images/sequences/non-email-steps.png)
</Frame>

As a reminder, these steps are treated as tasks and **will not** be automated by Juicebox. When it is time to complete a task, the task owner will receive a notification from Juicebox and will then need to complete it manually via LinkedIn or phone.

When a task has been triggered, it will appear on your home page. From here, you can execute your task or filter by status, task type, or project!

<Frame>
  ![Home page task list with filters for status, task type, and project](/images/sequences/tasks-home.png)
</Frame>

After completing the task, you can **Mark as completed** or **Skip** to resume the sequence for any remaining steps. Note that when a task is pending, the sequence will be paused until the task is completed.

<Frame>
  ![Mark as completed and Skip options for a pending task](/images/sequences/complete-task.png)
</Frame>

## Pausing, resuming, or stopping a sequence

If you want to stop, pause or resume a sequence, you can select the individual sequence from the Sequences homepage. Once that sequence has been selected, you can bulk select all candidates next to **Full name** or individually select the candidates you want to stop/resume the sequence for.

<Frame>
  ![Candidate selection within a sequence for pausing, resuming, or stopping](/images/sequences/pause-resume-stop.png)
</Frame>

Sequences can be paused or stopped in a few different ways:

- **Automatically**
  - When a candidate replies to your outreach, the sequence will automatically stop for that candidate.
  - If the mailbox is disconnected from Juicebox, all sequences from that mailbox will automatically pause.
- **Manually**
  - You can manually **pause**, **resume**, or **stop** a sequence at any time, either for a single or all candidate.

Manual controls are also helpful if you want to delay outreach, make edits, or stop follow-ups after receiving a response through another channel (like LinkedIn).

1. **Pause**: Temporarily halts all remaining steps in the sequence. Use this if you want to resume sending later.
2. **Resume**: Continues the sequence from the next unsent step for each candidate.
3. **Stop**: Permanently ends the sequence for the selected candidate(s). No further steps will be sent. Use this when a profile is no longer relevant, has opted out, or has already engaged outside of email.
